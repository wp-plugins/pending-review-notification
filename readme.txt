Pending Review Notification on WordPress Dashboard.

== Description ==

Show a Dashboard Widget if Pending Review Posts available.

== Installation ==

1. Upload `prn.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
